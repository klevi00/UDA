# rda-portal

L’obbiettivo dello stage è la realizzazione di un applicativo per la gestione delle RDA (Richieste d’acquisto). 
Durante la prima settimana verrà fatta formazione mentre per le restanti due settimane verrà sviluppato il codice che permetterà la gestione delle RDA.


## Struttura del progetto

/sql -> Cartella che contiene gli script necessari al software
